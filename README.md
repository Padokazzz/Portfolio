# Portfolio

Portfolio profissional construido com Next.js, App Router, TypeScript, Tailwind CSS e shadcn/ui.

## Stack

- Next.js com App Router
- React
- TypeScript
- Tailwind CSS v4
- shadcn/ui com Radix e preset Nova
- Framer Motion
- Lucide React

## Desenvolvimento

Instale as dependencias:

```bash
npm install
```

Rode o servidor local:

```bash
npm run dev
```

Acesse:

```txt
http://127.0.0.1:3000
```

## Qualidade

```bash
npm run lint
npm run build
```
